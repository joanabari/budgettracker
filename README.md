## Budget Tracker 

**What does this tool do:** [This tool has the purpose to budget your spendings, so you can see all of them at once, and it adds them up and divides them into diferent categories.]

**How to use it:** [It is a very easy and accesible tool, you just input the description, amount, date, and category of your purchase and then it will start saving all of them.]
